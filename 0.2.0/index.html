Overview

The Keypop Calypso Crypto Asymmetric C++ API defines interfaces for asymmetric cryptographic operations and PKI certificate management in Calypso cards.

  • keypop::calypso::crypto::asymmetric::certificate::spi::CaCertificateSpi CA certificate management

  • keypop::calypso::crypto::asymmetric::certificate::spi::CardCertificateSpi Card certificate management
  • keypop::calypso::crypto::asymmetric::certificate::spi::CardPublicKeySpi Card public key access

Certificate Parsing
